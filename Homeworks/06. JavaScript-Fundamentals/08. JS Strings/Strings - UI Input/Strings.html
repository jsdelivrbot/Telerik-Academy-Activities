Problem 4. Parse tags

  • You are given a text. Write a function that changes the text in all regions:
<upcase>text</upcase> to uppercase.
<lowcase>text</lowcase> to lowercase
<mixcase>text</mixcase> to mix casing(random)

Examples:

input result
We are <mixcase>living</mixcase> in a <upcase>yellow submarine</upcase>. We <mixcase>don′t</mixcase> have <lowcase>anything</lowcase> else. We are LiVinG in a YELLOW SUBMARINE. We dOn'T have anything else.

Note: Regions can be nested.

Problem 7. Parse URL

  • Write a script that parses an URL address given in the format: [protocol]://[server]/[resource] and extracts from it the [protocol], [server] and [resource] elements.
  • Return the elements in a JSON object.

Examples:

URL result
http://telerikacademy.com/Courses/Courses/Details/239 protocol: http, server: telerikacademy.com resource: /Courses/Courses/Details/239

Examples:

input output

Also visit <a href="www.devbg.org">our forum</a> to discuss the courses.</p> <p>Please visit [URL=http://academy.telerik.com]our site[/URL] to choose a training course. Problem 9. Extract e-mails

  • Write a function for extracting all email addresses from given text.
  • All sub-strings that match the format @… should be recognized as emails.
  • Return the emails as array of strings.

Some Examples (for easier check):

input output
